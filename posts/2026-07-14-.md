今天完成了博客的第一版部署。这个站点不需要服务器，也不需要数据库，只要把静态文件放到 GitHub 仓库里，再开启 GitHub Pages 就可以访问。

## 我完成了什么

- 创建 GitHub 仓库
- 把本地 `index.html` 推送到远程仓库
- 开启 GitHub Pages
- 做了一个 Sakura 氛围的静态博客首页
